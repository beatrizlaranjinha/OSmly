OSmly — Simulador de Gestão de Processos, Escalonamento e Gestão de Memória em OCaml



Como executar ? 


```
dune buil
dune exec
```


build:
```bash
	dune build
```

run:
```bash
	dune exec group_project -- fcfs
```

fcfs:
```bash
	dune exec group_project -- fcfs
```

priority:
```bash
	dune exec group_project -- priority
```

sjfs:
```bash
	dune exec group_project -- sjfs
```

rebuild:
```bash
	dune clean && dune build
```

clean:
```bash
	dune clean
```

testfork:
```bash
	dune exec group_project -- fcfs
```
